# Security Policy

This repository contains the public website for NIWO Systems. If you find a security issue related to the website, please report it privately so it can be reviewed before any public discussion.

## Reporting a Security Issue

Please send security reports to:

- `security@niwosystems.com`

You can also find security contact information at:

- `https://www.niwosystems.com/.well-known/security.txt`

Include enough detail to understand and reproduce the issue, such as the affected URL, expected behavior, observed behavior, and any safe reproduction steps. Do not include sensitive personal data, client data, credentials, or information from systems you are not authorized to test.

Please do not disclose the issue publicly, open a public GitHub issue, or share exploit details before it has been reviewed.

## Scope

This policy is for security issues affecting the NIWO Systems website and repository configuration. It does not authorize testing against third-party systems, client systems, or infrastructure outside the website's public surface.

Reports are reviewed as availability allows, with priority given to issues that could affect confidentiality, integrity, availability, or visitor privacy.

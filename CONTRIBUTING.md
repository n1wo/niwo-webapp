# Contributing

This repository is primarily maintained by NIWO Systems as the public website for an independent cybersecurity business. It is public for transparency and technical credibility, but it is not intended to operate as a large open-source community project.

## Small Changes

Small typo fixes, documentation corrections, broken-link fixes, and similarly focused improvements may be considered through issues or pull requests.

## Larger Changes

For larger changes, design changes, new sections, workflow changes, or anything that affects site positioning, please open a short discussion first. This helps keep the website aligned with the business context and avoids unnecessary work.

## Sensitive Information

Do not include sensitive client information, private business details, credentials, vulnerability details, or personal data in GitHub issues, pull requests, commit messages, or screenshots.

Security issues should be reported privately according to [SECURITY.md](./SECURITY.md), not through public GitHub issues.
